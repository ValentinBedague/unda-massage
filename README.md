# Unda – Site vitrine

Site vitrine statique en HTML/CSS/JS avec :

- GSAP pour les animations
- Barba.js pour les transitions entre pages
- Lenis pour le scroll fluide

## Structure

- `index.html` – Page d’accueil
- `massages.html` – Page « Les massages »
- `a-propos.html` – Page « À propos »
- `reserver.html` – Page « Réserver »
- `assets/css/style.css` – Styles globaux
- `assets/js/main.js` – Initialisation GSAP, Barba.js et Lenis

## Lancer le site en local

Ouvrez simplement `index.html` dans votre navigateur, ou servez le dossier avec un petit serveur local pour de meilleurs résultats avec Barba.js (par exemple `npx serve` dans ce dossier si Node.js est installé).
